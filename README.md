# Installing on devices
## Windows
• Open Google Chrome
• Click three dots
• Press cast, save and share
• Install
## MacOS
• Open Google Chrome
• Click three dots
• Save and share
• Install page as app
## Ios
• Open website
• Press PESTOhome Ipa file
• And install
• (Note: you will need to sideload it or you could use method 2 which is to open Safari, click share, click view more, and click Add to Home Screen ) 
## Android
• Open website
